# snowfall

*a quiet instrument*

---

generative audiovisual system  
inspired by weather, not performance

open it  
start it  
let it run

---

```
    ▪ density    how many
    ▪ size       how large  
    ▪ wind       how far
    ▪ mood       what scale
    ▪ dither     what texture
```

---

```
    space     play / pause
    h         hide sidebar
    d         drums
    t         theme
    r         reseed
```

---

each flake may become a note  
not every note is obvious  
not every flake is heard

motion → sound  
density → harmony  
time → evolution

---

**drums**

```
    ▪ 16 steps × 6 voices
    ▪ euclidean / random / glitch fills
    ▪ evolve mode — pattern mutates each loop
    ▪ stutter pad — rate × chance
    ▪ tone pad — filter × decay
```

idm-friendly  
click and drag

---

```
npm install
npm run dev
```

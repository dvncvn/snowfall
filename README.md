# snowfall

*a quiet instrument*

---

generative audiovisual system  
inspired by weather, not performance

open it  
start it  
let it run

---

```
    ▪ density    how many
    ▪ size       how large  
    ▪ wind       how far
    ▪ mood       what scale
    ▪ delay      what echoes
```

---

```
    space     play / pause
    h         hide
    r         reseed
    p         pixel
    d         dark / light
```

---

each flake may become a note  
not every note is obvious  
not every flake is heard

motion → sound  
density → harmony  
time → evolution

---

```
npm install
npm run dev
```
